# 🎨 Frontend - База данных пользователей

Веб-интерфейс для просмотра данных пользователей телеграм-бота.

## 📁 Структура

```
frontend/
├── index.html      # Главная страница
├── css/
│   └── style.css   # Стили
└── js/
    └── app.js      # JavaScript логика
```

## 🚀 Использование

Сервер автоматически подает эти файлы на `http://localhost:5000`

### Запуск сервера:
```bash
python3 app.py
```

### API:
- `GET /` — HTML страница
- `GET /css/style.css` — CSS стили
- `GET /js/app.js` — JavaScript
- `GET /api/users` — JSON данные пользователей

## ✨ Возможности

- ✅ Красивый интерфейс с таблицей
- ✅ Автоматическое обновление каждые 5 сек
- ✅ Кнопка ручного обновления
- ✅ Счётчик пользователей
- ✅ Время последнего обновления
- ✅ Обработка ошибок

## 📱 Отображение

Таблица показывает для каждого пользователя:
- ID в базе
- Имя
- Телефон
- Дата регистрации
