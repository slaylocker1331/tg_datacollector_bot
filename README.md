# 🤖 Telegram Data Collector Bot

Полнофункциональное приложение для сбора данных пользователей через Telegram бот с веб-интерфейсом для просмотра собранных данных.

## ✨ Возможности

- 🤖 **Telegram Bot** - автоматический сбор контактов пользователей
- 🌐 **Web Server** - REST API для получения данных
- 📊 **Web Dashboard** - красивый интерфейс для просмотра пользователей
- 💾 **SQLite Database** - безопасное хранение данных
- 🛡️ **Full Error Handling** - обработка всех ошибок сети и БД
- 📝 **Logging** - детальное логирование всех операций
- 🔄 **Auto-Reconnect** - автоматическое переподключение при разрыве
- 🔐 **Security** - защита от path traversal, валидация данных

## 📋 Требования

- Python 3.8+
- pip

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r backend/requirements.txt
```

### 2. Конфигурация

Скопируйте `.env.example` в `.env` и заполните токен:

```bash
cp .env.example .env
# Отредактируйте .env и вставьте ваш Telegram токен
nano .env
```

**Как получить Telegram токен:**
1. Напишите боту [@BotFather](https://t.me/botfather)
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в `.env`

### 3. Запуск

```bash
python3 main.py
```

Приложение запустится и выведет адреса:
- 📊 Веб-интерфейс: `http://localhost:5000`
- 🤖 Telegram бот: активен и слушает команды

Главная страница с таблицей пользователей

### GET `/api/users`

```json
{
  "status": "success",
  "count": 5,
  "data": [
    {
      "id": 1,
      "telegram_id": 123456789,
      "name": "Sam",
      "phone": "+1234567890",
      "created_at": "2026-04-24 14:30:45"
    }
  ]

```

## 🤖 Telegram Bot Commands

### `/start`
Начать взаимодействие с ботом. Бот запросит ваш контакт.

Отправьте контакт → бот сохранит его в БД → таблица обновится.

## ⚙️ Конфигурация (.env)

```dotenv
# Токен бота из @BotFather
TELEGRAM_TOKEN=your_token_here

# Настройки сервера
SERVER_HOST=0.0.0.0
SERVER_PORT=5000

# База данных
DATABASE_PATH=users.db

# Логирование
LOG_LEVEL=INFO
```

## 📊 Функции базы данных

- `init_db()` - инициализация БД
- `save_user(telegram_id, name, phone)` - сохранить/обновить пользователя
- `get_user(telegram_id)` - получить пользователя по ID
- `get_all_users()` - получить всех пользователей

## 🛡️ Безопасность

✅ **Защищено от:**
- Утечек токена (используется `.env`, исключены из Git)
- SQL injection (используются параметризованные запросы)
- Path traversal (проверка пути в API)
- Некорректных данных (валидация номеров, имён и т.д.)
- Сетевых сбоев (автоматическое переподключение)

## 🐛 Решение проблем

### Ошибка: "Invalid token"
- Проверьте, что токен правильно вставлен в `.env`
- Убедитесь, что это реальный токен из @BotFather

### Ошибка: "Port 5000 is already in use"
- Измените `SERVER_PORT` в `.env` на другой порт
- Или завершите процесс, занимающий порт 5000

### Бот не отвечает на команды
- Проверьте логи - там будут детали ошибки
- Убедитесь, что интернет соединение работает
- Бот может потребовать несколько секунд для переподключения
